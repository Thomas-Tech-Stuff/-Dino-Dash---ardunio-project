# -Dino-Dash---ardunio-project
A arduino version of the google dinosaur game. Made using a Arduino, LCD screen, button and some other components.

Why did I decide to make this ?
I decided to do this project to grow my knowledge of how LCD screens work and their applications so I can apply this knowledge to other bigger projects.

Adjustments I would make to this in the future:
